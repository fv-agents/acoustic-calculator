# Lumenear Akoestische Calculator

Indicatieve berekening van nagalmtijd (RT60) met het akoestische effect van [Lumenear](https://lumenear.com) producten.

**[→ Open de calculator]()**

## Wat doet het?

1. Voer ruimteafmetingen in (L × B × H)
2. Selecteer materialen (vloer, wanden, plafond)
3. Kies Lumenear producten
4. Zie direct de nagalmtijd voor en na, met beoordeling

## Methode

- **Sabine-formule**: RT60 = 0.161 × V / A
- **Spraakgemiddelde**: 500 / 1000 / 2000 Hz
- **+12% diffractie-bonus** op hangende producten
- **89 producten** uit de Lumenear 2026 catalogus

## Bronnen

- [Acoustic.ua](http://acoustic.ua) — absorptiecoëfficiënten
- Sengpiel Audio — kruisvalidatie
- Peutz rapport A 3432-1-RA (EN-ISO 354:2003)
